LICENSES
Frames includes a variety of open-source libraries. A comprehensive list:
boost - used in the Frames core
Don Clugston's Fast Delegate library - used in the Frames core
freetype - used in the Frames core
libjpeg - used in the Frames core
libpng - used in the Frames core
tinyformat - used in the Frames core
zlib - used in the Frames core
glew - used only for the OpenGL renderer
google test - used only for the test suite
SDL2 - used only for the test suite
lua - not currently used
All these libraries have open-source licenses which are compatible with commercial software. Some of them require references in your project's documentation to comply with the license. We strongly recommend consulting with your legal department to ensure you've fulfilled all your legal requirements.
Our long-term intent is that Frames should not a have mandatory dependency on any open-source library. If there's a library we depend on that you need extracted for legal or engineering reasons, please contact us.
The license information below is intended for the distribution of Frames. It may function as a good starting point for your own licensing requirements, but we make no assurances.
